# SmilodenGen

A Smilodon pride simulation game inspired by ClanGen.

## Features

- Dynasty leadership system
- Genetic coat system
- Mutations
- Multi-biome gameplay
- Patrols and hunting
- Mentorship system
- Pride politics
- Dynamic events
- Saving/loading
- Seasonal survival mechanics
